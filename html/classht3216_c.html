Detailed Description

This is the class for the ht3216C.

this class gives full control over the ledmatrix when writing data. with these functions the matrix can be enabled, disabled, written or cleared etc.

void ht3216C::initialize ( )

This function initializes the full ht3216C system.

In this function the SYS_EN command is executed after that the LED_ON, BLINK_OFF, MASTERMODE and COMOPTION. It also resets the RAM data on the ht3216C.
