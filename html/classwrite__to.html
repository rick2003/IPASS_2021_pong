Detailed Description

This class is a temporary class only used to write a sequence of bits to the ht3216C.

When this class is called, it instantly set the Chip Select pin low. after the class is called, the data can be written by using the writeData() function. When this class is done with writing the destructor sets the Chip Select pin high.

Attention
The destructor needs to be called or automatically activate by leaving a scope!

This function is used to write the actual data to the ht3216C.

This function runs bit by bit through the data The data pin is set high when the current bit is a 1 and a 0 if the current bit is low. For every bit the write pin is pulsed.

Parameters
numberis the number of bits in data. (unsigned int, 8bits)
dis the actual data that needs to be send. (unsigned int, 16bits)
Note
as long as the destructor is not activated this function can be called again.
